# LuaIntent — 1-Minute Demo Guide

This is a quick, interactive walkthrough you can use to demonstrate the power of the LuaIntent algorithmic engine live.

---

## 0) One-time setup (local)

Open your terminal and build the local bundle:

```bash
npx esbuild demo/bundle-entry.js --bundle --outfile=demo/lua-intent.bundle.js
```

Open the file in your browser: `demo/index.html`

---

## 1) 1-Minute Demo Script (Talk Track)

**0:00–0:15 — The Morphing UI**

"This isn't just swapping out a background image. This is a true Dynamic Website."

*Action:* Click the **"LuaIntent Diagnostics"** button in the top right to open the dashboard.
*Action:* Click the 3 preset buttons at the bottom of the dashboard:

1. **Preset: Buyer** -> Watch the entire layout shift into a 2-column focused checkout page.
2. **Preset: Reddit** -> Watch the layout morph perfectly into a 50/50 split-screen comparison table.
3. **Preset: Twitch** -> Watch the layout transition into an immersive, dark-mode gaming aesthetic.

**0:15–0:35 — Real-World Organic Inputs**

"But it doesn't just rely on preset buttons. Let's feed the algorithm real, organic data."

*Action:* Scroll up in the Diagnostics panel to the **Organic Inputs** section.
*Action:* Type `?q=cheap+affordable` into the URL box.
*Action:* Click **Run Algorithm Pipeline**.
*Watch:* The system instantly detects the "Budget/Buy Now" intent.

**0:35–1:00 — Explainable AI (The Terminal Logs)**

"Under the hood, we aren't sending this to a slow LLM. We are using strict mathematical fusion."

*Action:* Point to the **Live Streaming Matrix Logs** terminal box.
*Explain:* Show them exactly how the algorithm ran in real-time. Point out how `urlAnalyzer` extracted the keywords, how the `scoreFuser` applied the weights, and how `confidenceCalc` determined exactly how mathematically confident the system is before changing the UI.

---

## 2) If asked about Behavioral Tracking

"The system also listens to you."

*Action:* Leave the page open and slowly scroll up and down repeatedly.
*Explain:* The `behaviorAnalyzer` is constantly calculating your Engagement Score in the background. If you stay on the page longer and interact with the mouse, your Engagement Score goes up, actively serving as a multiplier for the algorithm's confidence!
